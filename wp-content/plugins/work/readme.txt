A custom post type plugin for managing portfolio work items.

== Description ==

Work is a simple plugin that creates a custom post type for managing portfolio work items. It allows you to showcase your work, projects, or jobs in a structured way.

Features include:

* Custom post type for work items
* Support for categories and custom taxonomies
* Hierarchical structure for organizing work items
* Full Gutenberg editor support

== Installation ==

1. Upload the `work` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start creating work items from the 'Work' menu in your admin dashboard

== Frequently Asked Questions ==

= Can I customize the URL structure? =

Yes, you can modify the rewrite rules in the plugin code.

= Does this plugin support custom taxonomies? =

Yes, it supports several custom taxonomies including murals, canvases, and design.
